# Topics

Each topic folder is a GitHub Pages review unit.

Topic status is tracked in `metadata.json`.

- `drafted`: created by `/write <topic>`
- `reviewed`: marked after human review
- `published`: marked after final publish approval

```text
topics/<topic-slug>/
  index.md
  metadata.json
  references.json
  visual_assets.json
  assets/
    workflow.mmd
```
